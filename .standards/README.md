## AI Agent Rules

1. **ALWAYS** reference relevant standards before coding
2. **ALWAYS** follow size limits when applicable
3. **ALWAYS** include tests where feasible
4. **NEVER** bypass documented workflows

## Key Principles

- Follow established patterns in the codebase
- Prefer clear, maintainable changes over clever ones
- Document non-obvious logic
